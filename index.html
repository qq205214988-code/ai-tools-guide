1目录站模板

优先搜：tool directory page design、beautiful bookmark page HTML、AI tools directory website template。最贴近你的用途。

2组件细节

参考 uiverse.io、TailwindCSS components、UI component library，用来优化按钮、卡片、标签和筛选栏。

3配色阴影

参考 coolors.co、cssgradient.io、shadows.brumm.af，控制颜色温暖但不花，阴影轻一点。
